# ⚔️ Bootloader Comparison

## 🪟 Windows Bootloader

* Closed source
* UEFI-based
* Highly abstracted

---

## 🐧 GRUB Bootloader

* Open source
* Multi-stage
* File system support
* Loads multiple OS

---

## 🚀 Ubaid Bootloader

* Fully custom
* Minimal
* Direct BIOS interaction
* Built for learning and control

---

## Key Difference

| Feature        | Windows | GRUB   | Ubaid |
| -------------- | ------- | ------ | ----- |
| Source         | Closed  | Open   | Open  |
| Complexity     | High    | Medium | Low   |
| Learning Value | Low     | Medium | High  |
